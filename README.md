# Creating a Caesar Cipher Program in Python

## Goal

Develop a program in python that can encrypt & decrypt messages using a caesar cipher.

## Capabilities

- Ability to encrypt or decrypt a given message
- Ability to encrypt/decrypt upper-case and lower-case letters
- User must enter the message to be encrypted/decrypted
- Can accept any shift value

## Programming Language

Python

## Example

Message to Encrypt: "Hello World"

Shift Value Used For This Example: 3

Encrypted Version Of The Above Message: "Khoor Zruog"

Decrypted Version Of The Above Message: "Hello World"
